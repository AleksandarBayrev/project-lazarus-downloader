# project-lazarus-downloader
Small utility to download from https://projectlazarus.bg/

## Requirements - latest .NET 10 Runtime, you can download it from [here](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)

## How to use
* Windows/Linux supported
* `project-lazarus-downloader.exe PATH\TO\JSON\FILE.json` with the urls of the magazines you want to download
* Example:
```json
[
    "https://projectlazarus.bg/ng-01/",
    "https://projectlazarus.bg/opm-01/",
    "https://projectlazarus.bg/pcc-01/",
    "https://projectlazarus.bg/cg-01/",
    "https://projectlazarus.bg/pcm-001/",
    "https://projectlazarus.bg/gw-01/",
    "https://projectlazarus.bg/im-01/",
    "https://projectlazarus.bg/mg-00/",
    "https://projectlazarus.bg/gnm-93-01/",
    "https://projectlazarus.bg/tg-92-03/"
]
```
